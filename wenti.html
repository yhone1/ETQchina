经常问的问题

运输
找不到答案？


找不到您要找的答案？您可以通过屏幕右下方的实时聊天联系我们，如果我们不在线，也可以通过电子邮件联系我们。

电邮我们
  • 地址和未送达的地块不正确


    • 如果您将包裹订购到错误的地址，包裹将退还给我们。收到并退款后，您可以重新订购商品。请注意，重定

      向的运费不予退还。


      这个答案有帮助吗？   是   否

  • 订单发货需要多长时间？


    • 所有在下午5点（CET）之前下达的订单将在同一个工作日发送。有关估计交货时间的概述，请参阅我们的

      发货页面上的图表。下订单后，我们会在结账时传达更准确的送货信息。


      这个答案有帮助吗？   是   否




订单处理
  • ETQ是否提供增值税折扣/扣除？


    • 是的，在欧盟航运区外，增值税将自动扣除。


      这个答案有帮助吗？   是   否

  • 我收到交易所多久了？

    • 交换的物品通常会在交付时的大致相同的天数内返回给我们。处理交换通常需要1-5个日历日，之后新产品将

      发送给您。一旦包裹转移到送货公司，ETQ 无法解释交货的任何延误。如有疑问，最好直接联系提供包裹的

      公司， 并为他们提供跟踪参考。


      这个答案有帮助吗？   是   否

  • 我是否支付欧盟以外的税收和关税？


    • 是的，确切的金额取决于国家和订购的商品。有关更多信息，您可以联系当地海关。请注意，ETQ不确定税


      费和进口费用。


      这个答案有帮助吗？   是   否

  • ETQ接受哪些付款方式？


    • 我们接受iDEAL，Creditcard（万事达卡，维萨卡），比特币，PayPal和Bankcontact / Mister Cash。


      这个答案有帮助吗？   是   否

  • 我该如何下订单？


    • 1）将您要购买的产品放在篮子里;


      2）继续退房以完成购买;


      3）进入结账页面后，填写所有必需的信息;


      4）选择付款方式;


      5）通过在支付服务提供商门户网站完成付款确认您的购买，之后将收到确认电子邮件。


      这个答案有帮助吗？   是   否

  • 我的订单状态是什么？


    • 您可以随时查看订单。为此：


      1）登录您的帐户;


      2）在“最近订单”下，选择您的订单;


      3）从这里，您可以查看订单详细信息并跟踪订单。


      您还可以通过确认电子邮件访问订单信息。


      此外，您将收到一封带有跟踪链接的电子邮件。


      这个答案有帮助吗？   是   否




交货
  • 我如何退货或换货？


    • 有关此问题的详尽答案，请参阅我们的退货政策。



      这个答案有帮助吗？   是   否

  • 下订单后我可以更改订单或地址吗？


    • 如果我们的货运代理尚未提取您的订单，我们只能对您的订单进行更改。请发送电子邮件至


      service@etq-amsterdam.com，并附上您的订单号和您想要进行的更改，例如更新的地址或不同的项目。



      这个答案有帮助吗？   是   否

      国，克罗地亚，塞浦路斯，捷克共和国，丹麦，萨尔瓦多，芬兰，法国，德国，英国，希腊，格林纳达，圭


      亚那，香港，匈牙利，冰岛，爱尔兰，意大利，日本，韩国（南），卢森堡，马来西亚，摩洛哥，荷兰，荷


      亚那，香港，匈牙利，冰岛，爱尔兰，意大利，日本，韩国（南），卢森堡，马来西亚，摩洛哥，荷兰，荷


      斯，瑞典，瑞士，台湾，美国。


      如果您不确定送货公司是否在您所在的地区运营，您可以随时通过service@etq-amsterdam.com与我们


      联系。


      这个答案有帮助吗？   是   否

  • 谁在送我的包裹？


    • 我们将所有比荷卢经销订单与PostNL，欧洲订单与DPD以及所有其他订单与FedEx一起发货。如果您想知道


      哪些运输选项适用于您的位置，请查看我们的运送页面。


      这个答案有帮助吗？   是   否

  • 交货费用是多少？


    • （国际）国家邮资费率取决于交货国家和订单的重量。 有关预计运费的更多信息，请参阅我们的运送页面上


      的概述 。


      这个答案有帮助吗？   是   否




账户
  • 我下了订单。下一步是什么？


    • 订单下达后，您将收到一封确认电子邮件。如果您未在1个工作日内收到电子邮件，请检查垃圾邮件收件箱或


      通过service@etq-amsterdam.com与我们的客户服务部门联系。订单发货后，您将收到一个跟踪链接，以


      便在线跟踪您的订单。


      这个答案有帮助吗？   是   否

  • 下订单后我可以更改订单吗？


    • 这取决于具体情况。所有在下午5点（CET）之前下达的订单将在同一天内处理。除周末外。一旦包裹被我们


      的货运代理人取走，我们无能为力。 如果您的订单有问题，您可以随时发送电子邮件至


      service@etq-amsterdam.com或通过在线聊天与我们联系，我们将尽力为您提供帮助。直接并为他们提供


      跟踪参考。


      这个答案有帮助吗？   是   否

  • 我需要一个帐户来下订单吗？


    • 在结账菜单中，您可以选择您想要创建帐户的天气。拥有一个帐户可以更轻松地跟进任何已完成的订单以及


      进行任何未来的订单。


      这个答案有帮助吗？   是   否

  • 我忘记了我的密码。我该怎么办？


    • 如果您忘记了密码，可以通过此链接请求重置密码。



      这个答案有帮助吗？   是   否
